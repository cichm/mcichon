Społeczność GitHub!

GitHub jest darmowym serwisem internetowym, umożliwiającym między innymi darmowe hostingowanie plików. Ponadto posiada funkcje, jakich nie posiada system git, a są nimi np rozbudowane statystyki, forki repozytoriów, organizacje oraz web hooks.
